# 🤖 Rule-Based AI Python ChatBot

A simple Command Line Interface (CLI) based AI ChatBot built using Python.  
This chatbot responds to predefined user queries using a rule-based approach.

---

## 📌 Project Overview

This project demonstrates basic AI logic using:
- Conditional Statements
- Dictionary-based response system
- Functions
- Loops
- Date & Time module

The chatbot greets the user based on the current time and answers basic questions.

---

## 🚀 Features

✅ Time-based Greeting (Morning / Afternoon / Evening / Night)  
✅ Predefined Question-Answer System  
✅ Continuous Chat Loop  
✅ Exit Command using `bye`  
✅ Simple and Beginner-Friendly Logic  

---

## 🛠️ Technologies Used

- Python
- datetime module
- Dictionary Data Structure
- Functions
- While Loop
